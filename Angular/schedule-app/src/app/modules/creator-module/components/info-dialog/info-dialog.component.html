Инструкция по добавлению Вы уже дали имя новому расписанию.
Чтобы изменить, нажмите на edit рядом с именем.
Чтобы начать добавлять пары в новое расписание, можно нажать кнопки "Добавить пару" или "Добавить окно" под любым из дней в расписании.

Некоторые другие сочетания клавиш:
  • Enter по завершении ввода текущей пары фиксирует текущую пару и переходит ко вводу следующей
  • Shift + Enter по завершении ввода текущей пары фиксирует текущую пару и добавляет окно
  • Ctrl + Enter по завершении ввода текущей пары фиксирует текущую пару и добавляет новую пару в следующем дне
  • Ctrl + Shift + Enter по завершении ввода текущей пары фиксирует текущую пару и добавляет новое окно в следующем дне
